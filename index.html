General Use & Information

Google Chrome is the recommended browser when using this application. However, other browsers may be used.

Get started:

  1. Add students in the class list sidebar menu
  2. Create your seating chart by adding one or more of the colored seats
  3. Randomize the seating arrangement
  4. If necessary, swap students using the 'Manage Students' mode
  5. Save your session

Note: This application saves sessions to your computer and is specific to the browser you are using. To load a session, you must be using the same browser on the same computer which the session was saved to. You can export sessions and reimport them into a different browser or computer.

Tips

  • Create a template. If you are going to use a similar layout for multiple seating charts, create the chart without adding the students. This way, you can easily reuse the same layout for multiple classes.
  • If you do not need the carpet seating chart, you can use it as another normal seating chart.
  • Export/Import sessions (under the 'Settings' tab) to use your sessions on a different computer.
